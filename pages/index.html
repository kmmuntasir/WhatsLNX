WhatsLNX

A minimalist, high-performance unofficial WhatsApp desktop client for Linux with Wayland support, native notifications, audio/video calling, and screen sharing.

From Source

Install and update WhatsLNX via APT on Ubuntu, Debian, Linux Mint, and Elementary OS.

One-time setup

Add the repository signing key:

shellcurl -fsSL https://kmmuntasir.github.io/WhatsLNX/whatslnx-archive-keyring.gpg \
  | sudo gpg --dearmor -o /usr/share/keyrings/whatslnx-archive-keyring.gpg

Add the repository:

shellecho "deb [arch=amd64 signed-by=/usr/share/keyrings/whatslnx-archive-keyring.gpg] \
  https://kmmuntasir.github.io/WhatsLNX stable main" \
  | sudo tee /etc/apt/sources.list.d/whatslnx.list

Update the package index:

shellsudo apt update

Install

shellsudo apt install whatslnx

Upgrade

shellsudo apt update && sudo apt upgrade whatslnx

Uninstall

shellsudo apt remove whatslnx

# Optional: remove the repository
sudo rm /etc/apt/sources.list.d/whatslnx.list
sudo rm /usr/share/keyrings/whatslnx-archive-keyring.gpg
sudo apt update

Remove config data

shellrm -rf ~/.config/whatslnx

Universal, no-install release. Works on any Linux distribution.

Download and run

shell# Download the latest AppImage
wget https://github.com/kmmuntasir/WhatsLNX/releases/latest/download/WhatsLNX-*.AppImage

# Make it executable
chmod +x WhatsLNX-*.AppImage

# Run
./WhatsLNX-*.AppImage

Uninstall

shellrm WhatsLNX-*.AppImage
rm -rf ~/.config/whatslnx

Build from source. Requires Node.js 24+ and npm.

Prerequisites

  • Node.js 24+ and npm
  • Linux (Ubuntu 24.04+ recommended)

Build

shellgit clone https://github.com/kmmuntasir/WhatsLNX.git
cd WhatsLNX
npm install
npm start        # Run in development mode
npm run build    # Build all packages (AppImage + DEB)

Individual targets

shellnpm run build:appimage
npm run build:deb

Build artifacts go to dist/.

🔗 Deep Links

WhatsLNX registers as the system handler for whatsapp:// URIs. Supported formats:

uriwhatsapp://send?phone=1234567890
whatsapp://send?phone=1234567890&text=Hello
whatsapp://send?text=Hello

Clicking these links in any application will open WhatsLNX and navigate to the compose view.
